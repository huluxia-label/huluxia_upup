# huluxia_upup

葫芦侠三楼一键签到小工具

```
go get github.com/huluxia-label/huluxia_upup
```

在你编译好的 `upup.exe` 目录下创建一个 `配置.json` 或者 `conf.json` 文件, 填入你的账号密码(必须安装此格式..)

```json
{
  "username": "x@qq.com",
  "password": "124453456546",
  "token": ""
}
```

然后双击打开程序就可以签到了
